Space-Prototype

A 3d multiplayer space game prototype made in Godot. The game is about mining asteroids and shooting them back to earth to get money for upgrades. I learned a lot about networking and 3d game design while making this project. One of the biggest challenges was getting the networking to work with every player and entity in a way that is generic and easy to use. I'm proud of the work I did on this project and I hope to continue working on it in the future.
